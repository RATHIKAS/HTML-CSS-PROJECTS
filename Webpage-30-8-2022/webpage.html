Bulid Extensions

Extensions

You can create extensions that combine Zoho Desk's functionalities with those of third party tools to make work easier and more effective for the end users of Zoho Desk.You do not require extensive expertise or experience in programming to create extensions.Functional knowledge of HTML,CSS and Javascript can help you to a great extent.

Build Extension

  • Install the node.js runtime environment.

    Download node.js

  • Install theZET CLI tool to build,test and package the extensions.Run Command
  • Get started by creating an extension

Upload Extension

After packaging your extension,perform the following stpes to upload the zip file for review:

  • Go to Sigma website
  • Click the New Extension Button
